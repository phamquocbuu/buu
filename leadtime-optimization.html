Project Overview

The LeadTime Optimization project was a sophisticated initiative to revolutionize delivery route tracking and optimization for J&T Express Vietnam's extensive logistics network. As Tech Lead and System Architect, I designed and implemented a high-performance system capable of processing massive volumes of logistics events while providing intelligent route optimization across the entire delivery network.

This mission-critical system required innovative solutions to handle over 30 million events per day while delivering real-time route optimization and delay detection capabilities across 2,000+ post offices nationwide, fundamentally transforming operational efficiency and delivery performance.

Goals & Challenges

Project Goals

  • Create a system to track and optimize delivery routes, enhancing operational efficiency
  • Provide real-time monitoring of logistics performance and deviations
  • Enable intelligent decision-making for route planning and resource allocation
  • Improve delivery speed and reliability across the entire network

Key Challenges

  • Handling extremely high-volume events (30+ million events per day)
  • Ensuring rapid, accurate route optimization across extensive geographic coverage
  • Real-time tracking and delay detection for effective logistics management
  • Scaling algorithms to handle complex multi-node routing scenarios

Technical Solution

High-Level Architecture

Route Optimization Architecture

Microservices-based system with real-time event processing and intelligent routing algorithms

Technologies Used

Backend: NodeJS, Laravel
Database: MongoDB (sharding, replica), MySQL (replication)
Message Queue: self-built with NodeJS and MongoDB
Infrastructure: Ubuntu Server (on-premises)

Key Technical Approaches

  • Real-time data processing via message queues and microservices architecture
  • Custom route optimization algorithms for extensive network coverage
  • Horizontal scaling with optimize database sharding and replication strategies
  • Event-driven architecture for real-time logistics tracking and monitoring
  • Advanced caching mechanisms for high-performance route calculations
