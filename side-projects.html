Featured Projects

Vimate – Automated Video Creation Platform

Developed a sophisticated backend-driven video generation platform that enables users to create custom videos from templates by providing text, images, and media URLs. The system leverages Node.js and Remotion.dev for video rendering, with serverless infrastructure for scalability and performance.

Technologies Used

Backend: Node.js
Video Processing: Remotion.dev
Serverless: Cloudflare Workers
Infrastructure: Google Cloud Run
Queue System: Redis
Storage: Cloudflare R2
Database: PostgreSQL

Key Features

  • Asynchronous job queues with Redis for efficient video processing
  • Asset storage and management via Cloudflare R2
  • PostgreSQL for comprehensive job tracking and status management
  • Optimized for scalability and fast rendering turnaround
  • Template-based video generation with custom content injection
NexBack – Unsplash-powered New Tab Chrome Extension

Created a Chrome Extension that transforms the default new tab experience with high-resolution photos from Unsplash based on user-selected topics. The extension focuses on performance optimization and user experience with intelligent caching and offline capabilities.

Technologies Used

Platform: Chrome Extension API
Frontend: JavaScript, HTML, CSS
API Integration: Unsplash API
Storage: Chrome Local Storage

Key Features

  • Image preloading and intelligent local caching for performance
  • Topic preferences saved to local storage for personalization
  • Download and attribution links for user convenience
  • Graceful fallback handling when offline
  • Clean, minimalistic interface design
RexSound – Relaxing Sound App for iOS

Developed a minimalistic iOS app that helps users relax, focus, or sleep better by playing curated nature sounds, ambient music, and white noise. Built with React Native and Expo, featuring a clean user interface and comprehensive audio management capabilities.

Technologies Used

Framework: React Native
Platform: Expo
Payments: In-App Purchase
Authentication: Email-based Auth
Distribution: App Store

Key Features

  • Clean, intuitive UI designed for relaxation and ease of use
  • Offline playbook capability for uninterrupted experience
  • User-friendly sound mixing and customization options
  • Background audio support for multitasking
  • In-app purchase integration for premium sounds
  • Privacy-compliant user data handling and security
  • Regular updates based on user feedback and analytics
