Project Overview

TachDiaChi.vn was an innovative NLP-based address parsing system specifically designed for Vietnamese addresses. As Team Lead, I spearheaded the development of this high-accuracy system that revolutionized waybill processing for J&T Express Vietnam, enabling streamlined mass upload processes and intelligent address standardization.

This sophisticated system leveraged advanced natural language processing algorithms tailored for the complexities of Vietnamese address formats, achieving exceptional parsing accuracy while supporting high-throughput processing requirements essential for large-scale logistics operations.

Goals & Challenges

Project Goals

  • Develop a high-accuracy NLP-based address parsing system for Vietnamese addresses
  • Streamline mass upload processes for waybill labeling and processing
  • Create intelligent address standardization and validation capabilities
  • Enable rapid processing of large address datasets with consistent accuracy

Key Challenges

  • Achieving high accuracy (>95%) in parsing diverse Vietnamese address formats
  • Supporting high-throughput processing (5,000+ records in under 2 minutes)
  • Ensuring system scalability and reliability under heavy load conditions
  • Handling the complexity and variations in Vietnamese addressing conventions
